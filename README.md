## 🧩 Engineering Philosophy

```
"You can't improve what you can't observe."
```

- **Observability first** — instrument before you optimise
- **Everything as code** — infra, config, dashboards, alerts
- **Failure is expected** — design for resilience, not perfection
- **Shift left** — catch problems in pipelines, not production
